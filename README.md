# 📊 TechNova Sales Data Analysis

## 📌 Project Overview
This project was completed as part of the **TechNova Data Analytics Internship**. The objective is to analyze sales data, identify business insights, and visualize trends using Python.

---

## 🎯 Objectives
- Load and analyze sales data from a CSV file.
- Calculate total revenue.
- Analyze sales by category.
- Identify top-performing products.
- Analyze sales across different regions.
- Visualize insights using charts.
- Generate a pivot table summary.

---

## 🛠️ Technologies Used
- Python 3
- Pandas
- Matplotlib
- OpenPyXL

---
